## "=>"(ダブルアロー演算子) と "->"(アロー演算子)　

| 演算子 | 利用シーン |
| --- | --- |
| "->"(アロー演算子) | プロパティやメソッドにアクセス |
| "=>"(ダブルアロー演算子) | 連想配列の定義<br>名前のない関数（無名関数、クロージャ） |
